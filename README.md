# Bivacco

**Bivacco** is a font designed by **Maria Vittoria Navati**.

This repository contains the public release materials for the typeface, including font files, documentation, and licensing information. The font is currently available for **personal use only**. Commercial use requires a separate license.

> **Version:** 1.000  
> **Release date:** June 2026  
> **Designer:** Maria Vittoria Navati  
> **Website:** [mariavittorianavati.com](https://mariavittorianavati.com)  
> **Contact:** [mariavittorianavati@gmail.com](mailto:mariavittorianavati@gmail.com)

## About

Bivacco is an independent type design project by Maria Vittoria Navati. This first release is intended for personal, educational, and non-commercial experimentation.

As an early release, the font may still evolve. Future updates may include refinements to spacing, kerning, glyph coverage, OpenType features, and file formats.
